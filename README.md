## What is this?

A daily HackerRank grind log — one problem at a time, every single day.

No shortcuts. No skipped days. Just steady reps building toward stronger fundamentals in AI/ML engineering and software engineering interviews.

Every solution committed here is a real solve — documented and pushed the same day.

## Repository Structure

```
hackerrank.cooking/
│
├── python/              <- Python warmups & problem sets
├── data-structures/     <- Arrays, linked lists, trees, heaps
├── algorithms/          <- Sorting, searching, DP, graphs
├── sql/                 <- Queries, joins, aggregations
├── ai/                  <- AI/ML track problems
│
└── update_progress.py   <- Auto-updates this README
```

<details>
<summary><b>How auto-progress works</b></summary>
<br>

After adding a new solution file, run:

```bash

```

The script scans each topic folder, counts solved files, and rewrites the progress table below automatically. Then commit and push.

</details>
